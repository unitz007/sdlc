# Installation

## Prerequisites

- Go 1.20 or later
- Git

## Installing from source

```bash
git clone https://github.com/unitz007/sdlc.git
cd sdlc
go install ./...
```

The binary will be installed to `$(go env GOPATH)/bin`.

## Installing via `go get`

```bash
go get github.com/unitz007/sdlc
```

This will download and install the `sdlc` command.
